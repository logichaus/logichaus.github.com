Training

We offer two different tracks for JRuby training. The first track is designed for Java developers who are to want to learn how to use Ruby in their applications. The second track is for developers with a Ruby background, and little knowledge of Java, who want to run their applications on the JVM.

JRuby for Ruby Developers (1 or 2 days)

The JRuby for Ruby Developers training is offered in 1 or 2 day course. In both courses developers will learn the following:

  • How to port MRI apps to the JVM.
  • How the JVM memory manager works.
  • How monitor a JVM process.
  • How to deploy applications with Warbler, Trinidad and TorqueBox.

The 2 day course will spend more time with the server frameworks, and how they can be used to simplify deployment.

JRuby for Java Developers (4 days)

The JRuby for Java Developers training is designed for programmers who already use Java but are not familiar with Ruby. It is offered as a 4 day course, in which students will learn about the Ruby langage and how to use it on the JVM. Topics include:

  • Basic Ruby language primer.
  • Using RedBridge and the Java Scripting APIs to embedded Ruby in Java.
  • How to deploy applications with Warbler, Trinidad and TorqueBox.
